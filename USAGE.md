# SAM 半自动数据标注工具 使用指南

## 环境准备

```bash
# 1. 激活 conda 环境
conda activate data_annotation

# 2. 确认 Python >= 3.10 且 PyTorch 可用 CUDA（可选但推荐）
python -c "import torch; print(torch.cuda.is_available())"
# 输出 True 表示 GPU 可用，False 则使用 CPU（较慢）
```

## 模式一：手动标注（无需加载模型）

### 步骤

**1. 打开图片目录**

点击工具栏 `...` 按钮，或菜单 `文件 → 打开目录`，选择图片所在文件夹。左侧文件列表会显示该目录下所有图片。

**2. 选择工具**

工具栏上点击对应工具，或使用快捷键：

| 工具 | 快捷键 | 用途 |
|------|--------|------|
| 选择 | S | 点击选中标注，拖动顶点编辑，拖动整体移动 |
| 多边形 | P | 点击描点，右键或双击闭合 |
| 矩形 | R | 按住拖拽绘制 |

**3. 绘制标注**

- **多边形**：左键点击放置顶点，依次点击形成轮廓，右键点击（或双击最后一点）闭合多边形
- **矩形**：左键按下拖动，松开完成

**4. 设置标签**

在右侧"标签列表"面板中点击 `+ 新建` 创建一个标签（如 "cat", "dog", "car"），然后点击该标签选中它。之后绘制的所有标注都会使用这个标签。

**5. 修改标注**

- 切换到 **选择工具 (S)**，点击任意标注选中它（顶点变为可拖拽的圆点）
- 拖动圆点修改形状
- 拖动标注中部移动整体位置
- 按 **Delete** 删除选中的标注

**6. 保存**

按 **Ctrl+S** 或菜单 `文件 → 保存`。标注会保存为与图片同名的 `.json` 文件，格式与 labelme 完全兼容。

**7. 切换图片**

按 **A** 上一张、**D** 下一张，或直接在左侧文件列表中点击。

### 快捷键速查

| 快捷键 | 功能 |
|--------|------|
| S | 选择工具 |
| P | 多边形工具 |
| R | 矩形工具 |
| Ctrl+S | 保存 |
| A | 上一张图片 |
| D | 下一张图片 |
| Delete | 删除选中标注 |
| Ctrl+滚轮 | 缩放画布 |
| 中键拖拽 | 平移画布 |
| F | 适应窗口 |

## 模式二：SAM2 点/框交互标注

> 适用场景：精细标注单个目标。用鼠标点几下就能得到分割结果。

### 步骤

**1. 切换到 SAM2 模式**

点击工具栏 `SAM2` 按钮。

**2. 加载模型**

在工具栏 SAM2 区域，选择模型规格：

| 规格 | 大小 | 速度 | 精度 | 推荐场景 |
|------|------|------|------|----------|
| tiny | 149MB | 最快 | 一般 | CPU / 低显存 |
| small | 176MB | 快 | 较好 | 快速预览 |
| **base_plus** | 309MB | 较快 | 好 | **日常标注（推荐）** |
| large | 857MB | 较慢 | 最好 | 追求精度 |

点击 `加载` 按钮。首次加载约需 5-15 秒（取决于磁盘速度）。

**3. 打开图片**

通过文件列表或菜单打开图片。加载图片时会自动计算 SAM2 的图像特征（embedding），状态栏显示"SAM2 embedding 已就绪"。

**4. 添加提示**

在图片上操作：

| 操作 | 效果 | 标记颜色 |
|------|------|----------|
| **左键点击** | 添加正点提示（这里是要分割的目标） | 绿色圆点 |
| **右键点击** | 添加负点提示（这里是背景） | 红色圆点 |
| **Ctrl+左键拖拽** | 添加矩形框提示 | 蓝色虚线框 |

每次操作后，画布上会**实时显示半透明青色 mask 预览**。

**5. 迭代修正**

如果 mask 不准确，继续添加正点/负点修正：
- 缺失区域 → 左键点击该区域
- 多余区域 → 右键点击该区域
- 按 **Backspace** 撤销最后一个提示点

**6. 确认标注**

满意后按 **Enter** 键。mask 会自动转为多边形标注，使用当前选中的标签。

按 **Esc** 取消所有提示，重新开始。

### 使用技巧

- 先用 **一个正点** 点击目标中心，观察初始 mask
- 如果 mask 包含了不该有的区域，用右键在这些区域点一下
- 如果 mask 缺失了部分区域，用左键在缺失区域补点
- 通常 **3-5 次点击** 就能得到高质量 mask

## 模式三：SAM3 文本提示标注

> 适用场景：一张图中批量找到所有同类目标。用自然语言描述想找什么，模型自动搜索。

### 步骤

**1. 切换到 SAM3 模式**

点击工具栏 `SAM3` 按钮。

**2. 加载模型**

选择模型规格 `sam3`（标准，3.3GB，一次加载约需 15-30 秒），点击 `加载`。

> 注意：SAM3 模型约 3.3GB，需要 6GB+ GPU 显存。如果显存不足，模型会自动使用 CPU（很慢）。

**3. 打开图片**

通过文件列表打开图片。

**4. 输入文本描述**

在右侧 **SAM3 文本提示面板** 的输入框中，输入**英文描述**：

| 想找什么 | 输入 |
|----------|------|
| 人 | `a person` |
| 猫 | `a cat` |
| 黑猫 | `a black cat` |
| 红色的汽车 | `a red car` |
| 戴眼镜的人 | `a person wearing glasses` |
| 桌子上的杯子 | `a cup on the table` |

按 **Enter** 或点击 `搜索`。

**5. 浏览候选结果**

SAM3 会返回多个候选 mask，在面板中按置信度从高到低排列：

```
找到 8 个候选（已过滤置信度 < 0.5）

☑ Mask #1  score: 0.92
☑ Mask #2  score: 0.87
☐ Mask #3  score: 0.73
☐ Mask #4  score: 0.51
```

- **画布上**：所有候选 mask 以半透明叠加显示
- **点击候选**：对应 mask 高亮，其他变淡
- **勾选框**：控制哪些候选最终被确认

**6. 调节置信度阈值**

拖动滑块 `置信度阈值` 过滤低置信度候选。阈值越高，结果越少但越可靠。

**7. 确认标注**

- `全选` / `全不选` 批量操作勾选框
- 最终勾选你想要的 mask
- 点击 `确认标注` → 所有勾选的 mask 转为多边形标注

## 常见问题

**Q: SAM2 加载模型时报错 "Config not found"？**

A: 确认 `Models\SAM2\` 目录下有对应的 `.pt` 文件，且 `third_party_repository\sam2\sam2\configs\` 目录下有对应配置。

**Q: SAM3 加载报错？**

A: 确认安装了 `triton-windows==3.3.0.post19`。如果 GPU 显存不足（需 6GB+），可以尝试 `sam3.1` 模型或使用 CPU（编辑 `sam3_predictor.py` 中 `device` 为 `"cpu"`）。

**Q: 保存的 JSON 能在 labelme 中打开吗？**

A: 可以。格式完全兼容。

**Q: 如何用中文标签？**

A: 在标签面板中直接输入中文标签名即可。但 SAM3 文本提示目前建议使用英文。

**Q: 如何切换模型而不重启？**

A: 直接在工具栏选择新规格，点击 `加载` 即可。旧模型会自动卸载释放显存。
